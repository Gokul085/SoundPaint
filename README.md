# SoundPaint

SoundPaint is a Cross-Modal Generative AI System that converts paintings into music by analyzing visual emotional content.

## Technologies Used
- Python
- NumPy
- Scikit-Learn
- Pillow
- PrettyMIDI
- FluidSynth

## Features
- Dominant Color Extraction
- Emotion Detection
- MIDI Music Generation
- Multi-Instrument Composition
